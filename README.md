# ✅ todo-app

A clean and simple Todo app built with React & TypeScript.
Create, complete, and manage your daily tasks with a minimal and responsive UI.

---

## ✨ Features

- ✅ Add and complete tasks
- 🗑 Delete tasks
- 📱 Responsive design
- 🎨 Styled with SCSS

```bash
cd todo-app
npm install
npm run dev
```
